# Exercicio - Aula Css: Box Model e Flexbox

- Box Model: margin, border, padding e content.
- Flexbox: alinhamento e distribuição de itens (justify-content, align-items, flex).

Como usar:

1. Abra o arquivo `index.html` no seu navegador.
2. Observe a seção "Box Model" para ver como margin, border e padding aparecem ao redor do content.
3. Observe a seção "Flexbox" para ver como os itens são distribuídos e como o item com `flex: 1` ocupa espaço adicional.
